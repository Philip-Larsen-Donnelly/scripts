# DHIS2 Release script

## Usage

```
./release.sh version [suffix]
```

## Examples

### Release 1.0.0

```
./release.sh 1.0.0 # gives you 1.0.0
```

### Release 1.0.0 RC1

```
./release.sh 1.0.0 rc1 # gives you 1.0.0-rc1
```

### Release 1.0.0 beta

```
./release.sh 1.0.0 beta # gives you 1.0.0-beta
```
